# Manual de Eclipse

Bienvenido al manual de Eclipse, la plataforma de gestión para tu estudio. Esta guía está diseñada para dueños de estudios, personal administrativo y profesores.

> ¿Buscas documentación técnica para desarrolladores? Consulta la [Developer Documentation](/guide/introduction) en inglés.
